## MLO 到底是什么来头？

MyLifeOrganized 是一款已经迭代了十几年的老牌 GTD 任务管理工具，目前最新版本为 6.x，支持 Windows 和 Android 平台（iOS 版本由官方在 App Store 单独销售）。

它最核心的设计理念是：**任务清单不是一个平铺的列表，而是一棵无限层级的树**。你可以把一个大目标拆成项目，项目拆成子任务，子任务还能再往下拆——直到每一个「叶子节点」都是一个清晰可执行的具体行动。

搭配上 MLO 内置的智能排序算法，软件会根据优先级、紧急程度、场景等属性，自动给你生成一份「现在该干什么」的精简行动清单。不用你手动排，它帮你想好了。

这个思路和 OmniFocus 很像——事实上，MLO 在 Windows 平台上的地位，某种程度上就是 Windows 用户口中的「平替 OmniFocus」，只不过价格要亲民得多。
